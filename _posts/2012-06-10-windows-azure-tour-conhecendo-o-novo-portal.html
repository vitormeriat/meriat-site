Nós redesenhamos o portal de gerenciamento. Clique em 'Next' para participar do passeio… Foi exatamente assim que fui recepcionado ao o consumir os novos recursos disponíveis para o Windows Azure.

Neste post pretendo fazer este tour mostrando um pouco das novidades do novo portal de gestão do Azure.

Como postei anteriormente em Por dentro do novo Windows Azure, o novo portal de gestão do Windows Azure trouxe algumas novidades, não só em questão estética mas em organização lógica.

Para iniciar o o portal está todo em HTML 5.O grande ganho vem pela maior fluidez de conteúdo, melhoramento nas pesquisas e velocidade de acesso. Para aproveitar a nova empreitada da Microsoft, o novo portal está com uma cara metro-styled muito legal.

O ponto positivo que mais chama a atenção é que agora o novo portal está funcional em todos os navegadores. Isto é realmente importante se você já tentou acessar o antigo portal por um iPad…

O novo portal é construído em cima de uma API de gerenciamento baseada no modelo REST dentro do Windows Azure, logo tudo o que você pode fazer através do portal também pode ser feito através de programação acessando esta Web API

 

Main Menu

Utilize este menu para explorar o site do Windows Azure, acessar os recursos do portal que você já está usando, mas não estão visíveis ou para simplesmente sair de sua sessão de portal.

Create New

Criar novos serviços, recursos e componentes rapidamente com o mínimo de configuração e então fazer facilmente o ajuste “fino” das configurações mais tarde.

Commands

Comandos globais e contextuais, tais como parar, configurar ou excluir fornecer o controle imediato sobre seus serviços, recursos e componentes.

 

Notifications

Exibe as notificações e os status de comandos ativos bem como a saúde de seus serviços em um piscar de olhos …

 

 

Organização lógica

 

O novo portal do Azure aponta para um novo modelo organizacional dos recursos mesclando as opções já existentes com os novos recursos.

Aqui poderemos ver todos o serviços ativos bem como acessar detalhada e especificamente o serviço selecionado.

Como mencionei no post anterior, já estão incluídos aqui os novos recursos:

  • Web Sites saiba mais aqui!
  • Virtual Machines saiba mais aqui!
  • Networks saiba mais aqui!

Um ponto que particularmente gostei foi o novo dashboard do painel. Agora  ao acessar um dos recursos temos dispostos no dashboard todas as opções para o recurso em questão. Esta nova “disposição” bem como a nova interface favorece a utilização do portal caso você tenha a necessidade de fazer um acompanhamento de algum recurso pelo seu celular, por exemplo. Ainda não testei no tablet mas no Windows Phone ficou show. Um exemplo do dashboard do recurso Web Site pode ser visto logo abaixo:

A visualização do monitor de recursos e relatórios está muito rápida, intuitiva e agradável.  O pouco tempo  que tive para testar e experimentar já deu para perceber o ótimo trabalho do time de Azure no portal. 

Na minha concepção houve um avanço nos quesitos de usabilidade e velocidade o quer permite uma maior facilidade para o acesso e uso em múltiplos dispositivos.

 

Nem tudo são flores

Como sempre nem tudo é perfeito e como podemos notar nem todos os recursos estão disponíveis. O novo portal ainda tem limitações e restrições o que nos leva a depender do antigo portal para determinadas funções. Ainda não existe (ou pelo menos não encontrei nada a respeito), uma data para que o novo portal esteja totalmente funcional.

 

Pessoal, espero que tenham gostado. Bom estudo a todos!
