Segue uma super dica para os azureiros de plantão, é um vídeo de Grant Holliday com o título: “From Server to Service: How Microsoft Moved TFS to Windows Azure”.

Este vídeo se trata de uma experiência em criar um Internet-scale service no Windows Azure. Ao invés de focar nos aspectos de codificação que são amplamente documentados no MSDN,  o autor concentra os esforços em aspectos operacionais que são cruciais para executar um serviço confiável. Neste vídeo temos dicas e idéias sobre o que é preciso para se disponibilizar um aplicativo existente o tornando um serviço escalável na internet por meio do Windows Azure. Aqui o autor faz a demonstração portando o Visual Studio Team Foudation Server, uma aplicação multi-tier cliente / servidor no Windows Azure.

Descrição no Chanel 9: Visual Studio Team Foundation Server is a multi-tier client/server application that Microsoft ported to Windows Azure. In this session, you will get an idea of what it takes to take an existing application and turn it into an Internet-scale service. Getting the code running is the easy part - you will learn about all the other things you need to think about when shifting from building a server to running a service, including deployment, patching, monitoring, scaling and testing.
