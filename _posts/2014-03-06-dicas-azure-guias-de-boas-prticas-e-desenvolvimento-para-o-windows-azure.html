Hoje vou postar uma dica valiosa: Uma compilação dos guias da Patterns & Practices (P&P) sobre Windows Azure.

Para quem ainda não conhece a P&P é um grupo focado em escrever documentações, padrões de arquitetura, melhores práticas e guias de desenvolvimento para soluções Microsoft. A ideia é trazer insumos para acelerar o desenvolvimento, reduzir riscos ao projeto técnico e direcionar a criação de soluções a nível escalar.

Os guias da P&P para Windows Azure visam cobrir desde migrar aplicações de um ambiente On-premises a desenvolvimento de soluções Big Data na Nuvem.

Indico fortemente que você siga o Patterns & Practices  nas mídias sociais para ficar interado das novidades, atualizações e lançamentos.
