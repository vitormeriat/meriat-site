Vamos iniciar nosso exemplo criando um projeto do tipo Console Application e lhe dando o nome de Builder. Agora vamos adicionar uma nova pasta com o nome de Model. Sua solução deve ficar como na figura 1.

Passo 1 - Product

Nosso primeiro passo será a criação do “Product” que neste caso será um veículo. Abra a Classe Montadora e modifique sua namespace para Model. Inclua o Namespace System.Collections para trabalharmos com coleções. Inclua o código da Listagem 1 como uma classe deste namespace.

   1: /// <summary>
   2: /// Produto
   3: /// </summary>
   4: public class Veiculo
   5: {
   6:     // Configura o tipo do Veículo
   7:     private string tipo;
   8:  
   9:     // Criamos uma Hashtable para armazenar as partes do veículo atribuindo a elas uma chave.
  10:     // Assim podemos realizar buscas nos elementos da coleção.
  11:     private Hashtable partes = new Hashtable();
  12:  
  13:     // Construtor da classe.
  14:     public Veiculo(string tipo)
  15:     {
  16:         this.tipo = tipo;
  17:     }
  18:  
  19:     // Crio um Indexador para a classe
  20:     public object this[string chave]
  21:     {
  22:         get { return partes[chave]; }
  23:         set { partes[chave] = value; }
  24:     }
  25:  
  26:     /// <summary>
  27:     /// Método responsável por imprimir os dados do Veículo
  28:     /// </summary> 
  29:     public void MostrarDadosDoVeiculo()
  30:     {
  31:         Console.WriteLine("n-----------------------------------------------------");
  32:         Console.WriteLine("Tipo do Veículo: {0}", tipo);
  33:         Console.WriteLine("Chassi : {0}", partes["chassi"]);
  34:         Console.WriteLine("Motor : {0}", partes["motor"]);
  35:         Console.WriteLine("Rodas : {0}", partes["rodas"]);
  36:         Console.WriteLine("Portas : {0}", partes["portas"]);
  37:  
  38:         Console.Read();
  39:     }
  40: }

 

Passo 2 - Abstract Builder.

A classe ConstruirVeiculo é o Abstract Builder, ou seja, ela apenas diz quais são os métodos que um Builder deve ter. Por ser abstrata, esta classe não pode ser instanciada. É preciso criar as heranças concretas que implementarão as funções introduzidas na super classe.

   1: /// <summary>
   2: /// Abstract Builder
   3: /// </summary>
   4: public abstract class ConstruirVeiculo
   5: {
   6:     // Crio um veículo
   7:     protected Veiculo veiculo;
   8:  
   9:     // Crio uma Propriedade
  10:     public Veiculo Veiculo
  11:     {
  12:         // Busco o veículo
  13:         get { return veiculo; }
  14:     }
  15:     // Métodos base.
  16:     public abstract void ContruirChassi();
  17:     public abstract void ConstruirMotor();
  18:     public abstract void ConstruirRodas();
  19:     public abstract void ConstruirPortas();
  20: }

 

Passo 3 - Concrete Builder

Concrete Builder são as classes que herdam de Abstract Builder. Cada classe concreta dessas é responsável por criar as etapas corretas do objeto a qual se referem.

Nossas classes Abstract Builder estão descritas na listagem 3. Note que que os métodos abstratos descritos na classe ConstruirVeiculo(nossa Abstract Builder), agora são implementados.

   1: /// <summary>
   2: /// Concret Builder - Construtor concreto 1.
   3: /// Constroi um veículo do tipo Moto
   4: /// </summary>
   5: class ConstrutorMoto : ConstruirVeiculo
   6: {
   7:     public override void ContruirChassi()
   8:     {
   9:         veiculo = new Veiculo("Moto");
  10:         veiculo["chassi"] = "Chassi da Moto";
  11:     }
  12:  
  13:     public override void ConstruirMotor()
  14:     {
  15:         veiculo["motor"] = "500 cc";
  16:     }
  17:  
  18:     public override void ConstruirRodas()
  19:     {
  20:         veiculo["rodas"] = "2";
  21:     }
  22:  
  23:     public override void ConstruirPortas()
  24:     {
  25:         veiculo["portas"] = "0";
  26:     }
  27: }
  28:  
  29: /// <summary>
  30: /// Concret Builder - Construtor concreto 2.
  31: /// Constroi um veículo do tipo Carro
  32: /// </summary>
  33: class ConstrutorCarro : ConstruirVeiculo
  34: {
  35:     public override void ContruirChassi()
  36:     {
  37:         veiculo = new Veiculo("Carro");
  38:         veiculo["chassi"] = "Chassi do Carro";
  39:     }
  40:

  49:     }
  50:

  52:     {
  53:         veiculo["portas"] = "4";
  54:     }
  55: }
