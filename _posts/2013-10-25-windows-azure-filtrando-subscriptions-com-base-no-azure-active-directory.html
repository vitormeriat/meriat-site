Como postei ontem (dia 24-10), existe um novo tipo de filtro para as subscriptions no portal do Windows Azure, o FILTER BY DIRECTORY. Confira na imagem abaixo:

Ontem ao logar no portal como faço todos os dias, precisei alterar meu filtro e logo percebi a seguinte URL:

O que ocorre é que desde o dia 22-10-2013 foi implementado no portal do Windows Azure o recurso de filtro baseado no Azure Active Directory.

Isso é interessante já que quando criamos uma assinatura, é criado um Microsoft Azure Directory que fica configurado com o nome da sua conta em um endereço tipo suaconta.onmicrosoft.com, que automaticamente vincula o usuário corrente. Essa e a estrutura que permite a adissão de co-administradores a sua subscription.

Este é um recuso que para mim faz muita diferença, já que tenho diversas contas em que estou vinculado, seja de clientes ou as minhas. Sem falar que é possível perceber um direciomanto da Microsoft quanto ao uso do Active Directory no Azure.
