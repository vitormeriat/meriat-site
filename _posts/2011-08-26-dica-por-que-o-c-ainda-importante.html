> Quais as vantagens a C++ mantenha sobre c#?

> Como o C++ se encaixar com a Web e as respectivas tecnologias?

> Quais são algumas das maiores alterações em C++ com a versão mais recente?

Estas são algumas das perguntas que o artigo de Keith Ward se propõe a elucidar. Este artigo que faz parte da MSDN Magazine de julho de 2011.

Neste artigo Keith fala casos onde o uso de C++ ainda faz sentido e como podemos perceber o C++ no mundo Web. Uma visão bem interessante e que traz algumas das novidades que podemos esperar para o C++. Leitura recomendada.

Em sua coluna fixa na MSDN Magazine, Keith aborda sobre a nova programação do Windows com C++ descrevendo como gerenciar os recursos da API do Windows de uma forma que seja natural para o desenvolvedor atual de C++.

Segue abaixo o link para o artigo mencionado:

http://msdn.microsoft.com/pt-br/magazine/hh288086.aspx

Um grande abraço e ótimo estudo!
