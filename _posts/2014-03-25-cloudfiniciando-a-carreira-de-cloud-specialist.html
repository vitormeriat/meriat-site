Olá pessoal. Apesar de estar no mundo das Nuvens “a long time ago”, os últimos meses (final de 2013 e início de 2014) estão trazendo novos caminhos para minha vida profissional.

Mudar para São Paulo e entrar em contato com uma gama mais diversificade de profissionais da TI tem sido escencial para expandir os horizontes, e uma das primeiras coisas que ouvi aqui foi: “Cara, você tem que conhecer os conceitos e não só a tecnologia. Para quem só conhece o martelo todo problema vira prego!”.

Foi nessa pegada que decidi estudar a tal da Cloud e não apenas uma tecnologia/player/fornecedor específico como Windows Azure ou Amazon Web Services…

Se você quer construir uma carreira como especialista em Cloud Computing, sugiro fortemente um plano de carreira baseado em:

  • Dominar os conceitos fundamentais
    • Certificações em vários sabores: Cloud Foundation, Cloud Architect…
  • Dominar uma área de atução
    • Dev – Infra – Manager
  • Dominar pelo menos um fornecedor
    • Windows Azure – AWS – IMB – GooglePataformCloud...

 

No caso das certificações existem uma série de orgãos certificadores. Os principais a nível de reconhecimento do mercado são:

  • CompTIA Cloud Essentials - saiba mais
  • Certified Cloud Professional by CloudSchool - saiba mais
  • EXIN's Cloud Qualification - saiba mais

Sugiro iniciar na ordem inversa. Todo o material que você precisa está disponível nos sites assim como quais são as provas disponíveis, títulos, quais as provas para uma determinada carreira e treinamentos credenciados.

Para passar nesta prova em questão eu adiquiri o Paper preparatório, segui a material base e pesquisei nas fontes citadas para ficar afiado no dia da prova.

Estas provas tem por objetivo atestar seu conhecimento nos conceitos, implementações, arquiteturas e outros “saberes” fundamentais para se implantar Cloud Computing em um ambiente corporativo.

 

Até mais e bom estudo a todos!
